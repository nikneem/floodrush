# FloodRush Gameplay Pillars

## 1. Time-pressure puzzle solving
Players should feel urgency from the incoming fluid, but the game must remain readable and strategic rather than chaotic.

## 2. Clear board language
Start points, finish points, fixed tiles, and placeable pipes must be visually and behaviorally distinct.

## 3. Deterministic rules
The same board state and timing inputs must always produce the same flow and score outcome.

## 4. Offline-first trust
Players should feel confident that progress is safe locally and will synchronize later without loss.

## 5. Meaningful progression
Released levels and global score comparison should make continued play feel rewarding without requiring social or monetization systems.
