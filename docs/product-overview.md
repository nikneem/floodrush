# FloodRush Product Overview

FloodRush is a landscape-only .NET MAUI puzzle game where the player races a spreading fluid across a grid. Each level starts with fixed board elements and one or more finish points. The player must place the right mix of pipe sections fast enough to guide the flow to every required destination.

The game is designed as an offline-first mobile experience. Players should be able to continue playing downloaded levels without an internet connection. When connectivity is available again, the game synchronizes player profile data, settings, scores, and newly released levels with the server.

The project includes both the game client and the server platform needed for player profiles, released level management, and worldwide score storage.

## Core promise
- Fast, readable puzzle gameplay with time pressure
- Deterministic game rules
- Landscape-first mobile presentation
- Offline-first progression with online synchronization
- Server-backed released levels and score comparison
